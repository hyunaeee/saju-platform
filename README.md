# 🪷 사주 플랫폼 (saju-platform)

> 한국 사주(사주명리학)를 웹에서 간단히 체험할 수 있도록 제작한  
> **사주 플랫폼 프로토타입 프로젝트**입니다.

---

## 📌 프로젝트 소개

**saju-platform**은 사용자가 입력한 생년월일 및 시간을 바탕으로  
사주 정보를 계산하고, 이를 웹 UI로 간단하게 보여주는 프로젝트입니다.

- 빠르게 아이디어를 검증하기 위한 **경량 웹 구조**
- 추후 AI 기반 사주 해석, 서버 연동, 사용자 저장 기능 등으로 확장 가능
- GitHub Pages 등으로 바로 배포 가능한 형태

---

## ✨ 주요 기능

- 📅 생년월일 및 시간 입력
- 🔢 사주 계산 로직 적용
- 📄 결과를 웹 화면으로 출력
- 🖥️ 별도 서버 없이 정적 페이지로 실행 가능
